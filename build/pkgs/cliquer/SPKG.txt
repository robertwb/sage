= Cliquer =

== Description ==
Cliquer is a set of C routines for finding cliques in an arbitrary weighted graph. It uses an exact branch-and-bound algorithm recently developed by Patr Ostergard.
Cliquer was mainly written by Sampo Niskanen, sampo.niskanenQiki.fi (Q=@).
